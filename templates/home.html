Many datasets have an ingest "pipeline" that requires multiple processing stages that run on various servers under different user IDs.This app provides a front-end to manage and monitor the processes as each dataset travels through the system.

The first project to test this system is "CMIP6-MOHC" (Met Office Hadley Centre climate simulations for the 6th Coupled Model Intercomparison Project). Select listings from the Views menu above to monitor existing datasets being processed. If you log in you will also be able to pause or resume the system and run other management functions.

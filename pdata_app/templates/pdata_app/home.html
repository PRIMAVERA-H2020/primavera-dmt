This is the PRIMAVERA Data Management Tool (DMT) to track data files produced for the PRIMAVERA project. PRIMAVERA is a Horizon 2020 project funded by European Commission. It is a collaboration between 19 European partners, led by the Met Office and the University of Reading.

To query the data that is available to users at JASMIN:

  • Variables Received Query

The PRIMAVERA data request is available:

  • View Variable Requests
  • View Data Requests

Requests to restore data from tape to group workspace can be viewed:

  • View Retrieval Requests

The following views are also available:

  • View Data Submissions
  • View Data Files
  • Outstanding Data Query
  • View CEDA Datasets
  • View ESGF Datasets
  • View Data Issues

Indicate that a data submission is complete:

  • Data Submission Complete
